# 变更：补充主页、设置和商店养成 Demo

## 为什么

现有 demo 已验证专注、能量、宠物成长和分享卡闭环，但用户希望继续推进到更完整的产品雏形：主页、设置页、专注后购买食物喂食、购买宠物形象等。该变更补齐基础养成和商店能力，让 demo 更接近可体验产品。

## 变更内容

- 将单页 demo 扩展为包含主页、商店、设置的底部导航结构。
- 使用专注获得的能量作为 demo 内货币。
- 增加食物商品，支持购买并喂食宠物。
- 增加宠物形象商品，支持购买并切换当前宠物形象。
- 增加设置页，支持每日提醒开关、柔和提示开关、声音反馈开关和数据重置。
- 更新 README，说明项目定位、运行方式、功能范围和验证命令。

## 不包含

- 不接真实支付。
- 不做云同步。
- 不做硬拦截。
- 不做 OHOS 检查。
