# 变更：融合 GitHub 同类产品洞察

## 为什么

已有 PRD 基于 PixPet、Focus Friend 和 Finch 的公开市场模式定义了 Focus Pet 的 MVP 方向。现在补充 GitHub 同类产品调研后，需要把 PomodoroCat、Study Buddy、Noto、WebPet、Buddy、桌宠和 AI 虚拟宠物等公开项目的可借鉴点整理成可执行产品策略，避免后续实现只停留在“像素宠物番茄钟”的表层。

## 变更内容

- 在市场调研中新增 GitHub 同类产品参考和融合原则。
- 在 PRD 中新增同类产品融合策略，明确哪些洞察进入 MVP、第二阶段和第三阶段。
- 强化 MVP 的即时奖励、真实努力货币、温和宠物状态和分享传播要求。
- 明确 Study Buddy 式目标系统、AI 伴侣和桌宠形态暂不进入 MVP。

## 不包含

- 不修改 Flutter 代码。
- 不新增付费随机奖励。
- 不新增硬拦截能力。
- 不复制任何同类产品的代码、素材、角色或文案。
