# 变更：像素玩具风首屏和宠物可爱化

## 为什么

当前 demo 功能可用，但首屏直接进入主页，默认给用户选择芽芽，缺少“自由三选一”的仪式感；宠物形象也偏硬，整体界面更像工具而不是可爱的养成产品。用户明确要求采用“方案 1：像素玩具风”，因此需要优化首次进入、宠物视觉和主页质感。

## 变更内容

- 首次进入先展示三只宠物大卡片，用户自由三选一。
- 用户选择宠物后再进入主页。
- 像素宠物矩阵改得更圆、更萌，加入耳朵、脸颊和更清晰的表情。
- 主页改成更像宠物房间的玩具风视觉。
- 商店和设置页跟随更柔和、更可爱的视觉语言。

## 不包含

- 不新增真实支付。
- 不接入硬拦截。
- 不新增后端或云同步。
- 不执行 OHOS 检查。
