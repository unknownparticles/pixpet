# 变更：实现 Focus Pet 可用 Demo

## 为什么

当前仓库已有产品调研、PRD 和 OpenSpec 产品基础，但还没有可运行应用。用户目标是“开发出一个可用 demo”，因此需要创建 Flutter iOS-first demo，用最小闭环验证 Focus Pet 的核心体验。

## 变更内容

- 创建 Flutter 项目骨架。
- 实现 soft-focus 专注计时流程。
- 实现初始宠物选择、能量奖励、等级成长和下一个解锁目标。
- 实现今日/本周统计、连续天数和完成次数展示。
- 实现分享卡预览。
- 使用生成式像素资产作为首版视觉素材。
- 使用临时测试验证核心逻辑，验证后删除测试文件。

## 不包含

- 不实现硬拦截。
- 不实现 OHOS 相关内容或检查。
- 不实现云同步、登录、社交、地图、战斗、NFC 或付费随机抽卡。
