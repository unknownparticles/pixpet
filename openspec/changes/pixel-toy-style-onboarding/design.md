# 设计：像素玩具风首屏和可爱化

## 首次进入

应用启动后先进入宠物选择界面。界面展示三张大宠物卡片，每张包含：

- 大号像素宠物。
- 宠物名称。
- 性格标签。
- 适合的学习场景。
- “选择它”按钮。

用户选择后，才进入主页、商店和设置。

## 视觉方向

- 背景使用柔和的天空/奶油色，而不是纯工具灰白。
- 卡片保持 8px 圆角，符合项目设计约束。
- 宠物区域像小房间或小舞台，有地毯、装饰、状态条。
- 色彩避免单一色系：使用青绿、蜜桃、奶油、薰衣草、暖黄共同构成玩具感。
- 不使用装饰性渐变球或漂浮光斑。

## 宠物形象

继续使用程序化像素宠物，但调整矩阵：

- 14x14 或更高信息密度。
- 更圆的头身比例。
- 三只宠物有明显颜色差异。
- 表情更可爱，包含眼睛、脸颊、嘴巴和小耳朵。

## 验证

- 临时 widget test 验证首次展示三选一，并且选择后进入主页。
- 运行 `flutter analyze`。
- 验证完成后删除临时测试文件。
