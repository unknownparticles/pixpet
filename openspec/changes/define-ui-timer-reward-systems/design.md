# 设计：UI、番茄钟和激励机制

## UI 设计

首版界面应把“宠物陪我学习”放在最高优先级。主页第一屏由宠物房间和番茄钟控制组成，底部导航只保留主页、商店、设置三项。商店和设置不能抢主页注意力，避免用户进入 App 后先被装饰、统计或复杂任务分散。

视觉方向采用像素玩具风：奶油底色、蜜桃强调、青绿成长、薰衣草休息和深棕文字。宠物房间要像一个小学习桌，而不是普通效率工具卡片。卡片圆角不超过 8px，保持清晰边界和轻量阴影。

## 番茄钟设计

番茄钟以 25 分钟为默认推荐，同时提供 15、45、60 分钟和 5 到 120 分钟自定义。MVP 不做复杂番茄循环，只保证未开始、进行中、暂停中、已完成、已放弃五个状态清晰。完成后优先展示奖励结算和宠物反馈；休息倒计时可以第二阶段再加入。

Soft-focus 的核心不是拦截，而是让用户愿意把手机放远。开始后进入沉浸式计时，放弃需要二次确认，离开 App 不做硬阻断，回到 App 后用温和反馈继续引导。

## 激励机制

激励机制围绕真实努力。能量来自完成专注，低价食物提供即时满足，中价装饰提供房间变化，高价宠物形象提供长期目标。奖励数值要让第一次专注后立刻接近一次喂食或小购买，前三个可见奖励应能在首日获得。

连续天数只做正向增强，不制造断签惩罚。饱腹感和心情用于表达陪伴，不用于死亡、生病或羞辱性反馈。
